Cài đặt API

Trạng thái:
API Keys đã lưu
Hướng dẫn lấy API Key

1. Truy cập https://openrouter.ai/keys

2. Đăng nhập hoặc đăng ký tài khoản.

3. Nhấn nút Create Key để tạo khóa mới.

4. Sao chép khóa và dán vào ô bên trên.

Lưu ý: Khóa thường bắt đầu bằng sk-or-v1-

API Key sẽ được lưu trong hệ thống, không bị lộ khi đẩy code lên GitHub.
